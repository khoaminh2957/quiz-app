Quick start

  1. Choose a language above (or press Alt+1..Alt+5).
  2. Start at the foundational stages: code reading → runtime errors → idioms → boundaries → truthy/falsy.
  3. Each stage has a mastery gate; clear it to unlock the next.
  4. Use flat quiz for random practice, or Progress to compare against the simulated cohort.

Why per-language tracks?

Lister 2004's multi-national study found that code-reading and tracing skills vary measurably by language. We partition the 1141 questions and 52 + 5 KCs into language tracks so practice stays focused on idioms and pitfalls of one language at a time — without losing the shared cross-language taxonomy (Koedinger 2012 KLI).
